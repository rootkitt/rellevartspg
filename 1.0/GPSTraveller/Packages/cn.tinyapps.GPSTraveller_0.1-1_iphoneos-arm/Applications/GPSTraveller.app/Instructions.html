1、穿越方法

　　穿越其实很简单，首先在地图上用手指长按所选择的穿越位置(选择后会看到一枚红色图钉掉落于此，并弹出一个包含该位置的地理信息的气泡)，然后点一下屏幕底部的开始按钮就可以了。

2、积分消耗算法

　　穿越所消耗的积分与所穿越的距离有关，请珍惜你获得的积分，利用好每一次穿越机会。下面是具体的积分消耗算法：

穿越距离<10公里 穿越距离>=10公里
不需要积分 每10公里消耗1积分

　　例如，如果要从广州穿越到香港星光大道，因为两地相距约130公里，就需要消耗约13个积分。

3、积分获取秘笈

　　积分是进行远距离穿越(>10公里)的必备行装。在现阶段，你需要通过下载应用来赚钱积分，但是有些应用的积分获取条件比较苛刻，请认真细看并严格遵守以避免浪费了宝贵时间，却没有获得积分。

　　在积分获取的页面，每一个应用旁边都有相关注释:在安装后，有些需要试用3分钟，有些需要注册账号并试用3分钟，有些应用需要创建游戏角色。当然，条件越多的应用，可能获取的积分也就越多。请尽量选择感兴趣的应用，或者条件比较宽松的应用进行下载并试用。

4、常见问题

  1. 穿越了有什么用？

    2. 借助微博、微信、米聊、陌陌等应用，启动穿越，可以让你的好友惊叹你日行千里的速度，可以向亲人好友撒个小谎，给个惊喜，也可以窝在床上，穿越到维多利亚港湾旁的夜色人群、巴黎的微醺咖啡厅、爱琴海的黑沙滩，偶遇异国他乡的每一个奇妙的缘分。穿越的更多乐趣，在于你去发掘。如果你发现了你的玩法很有意思，也请你在"设置"-"用户反馈"中反馈，分享给更多朋友。

  2. 为什么我启动后显示定位失败？

    3. 　　定位失败的原因可能是你的手机没有允许"GPS穿越器"访问你的位置信息，可能是你没有打开网络，请在“设置”中打开定位服务，并在里面设置“GPS穿越器”为可访问你的位置信息，并确认你的手机已联网。

  3. 下载了应用，积分却没有增加，为什么？

    4. 　　获取积分的条件其实大多比较严格，有些需要下载安装并打开试用一定时间（1分钟、3分钟、或5分钟)，有些需要进行账号注册，有些需要创建游戏角色。另外，下载你手机上已经存在的应用也是无法获取积分的。所以，请你认真查看每个应用所对应的要求，并严格遵守，避免浪费时间，却没有获得相应积分。

  4. 为什么穿越距离超过10公里，就要耗费积分？

    5. 　　首先，穿越的乐趣就在于不是每个人都可以轻松穿越。如果穿越太容易，太轻松，太多人玩穿越，那么穿越将失去新奇感，我们也会开始觉得无趣。我们希望让一部分人先穿越起来，并感受它的魔力。另外，为了保证提供长期优质的服务，我们需要有一定的经费来维继。当然，如有热心朋友能够捐助，请在"设置"-"用户反馈"中联系我们，我们将替所有穿越的朋友感激您。

  5. 如有其他问题，请在"设置"-"用户反馈"中提出。
